# Quine
Trabalho de ferramentas de CAD

A primeira informação a ser digitada é o número de váriaveis, aceitando valores de 1 a 26. (A a Z)

A segunda informação é o número da linha que possui 1.

Exemplo:

4	//A,B,C,D
1	//0000
2	//0001
3	//0010
4	//0011
5	//0100

Resultado
A'B' + A'BC'D'
